# Listas enlazadas

Las listas enlazadas son estructuras de datos datos en las cuales cada elemento tiene un puntero al próximo elemento de la lista.

Por ejemplo en un reproductor de canciones, como spotify, podemos considerar la lista de reproducción (playlist) como una lista enlazada. CAda canción tiene un puntero a la próxima, de modo que si usted le da click al botón de siguiente canción, 

|_Canción #1| -> |_Canción #2 | -> |_canción #3| ->
